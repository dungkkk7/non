# Hello World!

Chào mừng đến với hệ thống blog mới của tôi. Đây là bài viết mẫu đầu tiên sử dụng **Markdown**.

## Tại sao lại dùng Markdown?

1. **Dễ viết**: Không cần thẻ HTML rườm rà.
2. **Nhanh**: Tập trung vào nội dung.
3. **Đẹp**: Dễ dàng chuyển đổi sang HTML sạch.

## Code Example

Dưới đây là một ví dụ về code Python:

```python
def hello():
    print("Xin chào, Việt Nam!")

hello()
```
